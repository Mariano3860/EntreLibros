# Backlog EntreLibros

**Fecha de última actualización:** 14 de enero de 2025

## Resumen ejecutivo

EntreLibros es una plataforma web completa para el intercambio de libros entre usuarios, desarrollada con arquitectura moderna y escalable. El proyecto cuenta actualmente con **72 PRs mergeadas** que implementan funcionalidades core como autenticación JWT, mensajería en tiempo real, geolocalización de usuarios, integración con OpenLibrary API, chatbot basado en reglas, y documentación OpenAPI completa. El backend está construido en Node.js/Express con PostgreSQL y PostGIS, mientras que el frontend utiliza React/TypeScript con React Query y internacionalización. El proyecto incluye workflows de CI/CD, scripts de formateo automático, y una arquitectura de testing robusta.

## Matriz de Integración FE↔BE

### APIs del Backend existentes no integradas en Frontend

1. **Verificación de libros OpenLibrary** (`/api/books/verify`)
   - Búsqueda automática de información bibliográfica
   - Marcado de libros como verificados
   - **Acción**: Integrar en formulario de creación de libros

2. **Actualización de ubicación usuario** (`/api/user/location`)
   - Almacenamiento de coordenadas y radio de búsqueda
   - **Acción**: Crear interfaz de configuración geográfica

3. **Configuración de idioma** (`/api/user/language`)
   - Persistencia de preferencias de idioma
   - **Acción**: Validar integración con selector existente

### APIs del Frontend que requieren Backend

1. **Gestión de publicaciones**
   - CRUD para ofertas/solicitudes/intercambios de libros
   - **Backend faltante**: Controladores de publicaciones

2. **Gestión de conversaciones**
   - Persistencia de historial de mensajes
   - **Backend faltante**: Controladores de conversaciones

3. **Estadísticas de comunidad**
   - Métricas de usuarios, libros, intercambios
   - **Backend faltante**: Endpoints de estadísticas

4. **Rincones de libros**
   - Ubicaciones físicas para intercambio
   - **Backend faltante**: Controladores de rincones

5. **Sistema de notificaciones**
   - Push notifications, notificaciones internas
   - **Backend faltante**: Sistema de notificaciones
